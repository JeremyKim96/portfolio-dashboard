# 내 주식 포트폴리오 대시보드

해외주식 + 암호화폐 + 현금/예금을 한 화면에서 관리하는 로컬 도구.
사용자가 **단일 RAW 엑셀**을 폴더에 떨어뜨리면 자동으로 분석·시각화한다.

---

## 🚀 처음 1회 세팅

1. **Python 설치** (3.11 이상 권장)
   - https://www.python.org/downloads/ 에서 받기
   - 설치할 때 **"Add Python to PATH"** 꼭 체크!
2. **`run.bat` 더블클릭** — 가상환경과 패키지가 자동 설치됨 (수 분 소요)
3. 브라우저가 자동으로 열리면 끝.

## ⚡ 자동 시작 (이미 설정됨)

Windows 작업 스케줄러에 **PortfolioDashboard** 작업이 등록되어 있어서:
- PC 켜고 로그인하면 **자동으로 Streamlit + Cloudflare Tunnel이 백그라운드로 뜸**
- 콘솔창 안 보임 (pythonw.exe로 hidden)
- 언제든 **http://localhost:8765** 접속하면 응답

**수동 제어**:
- 시작: `Start-ScheduledTask -TaskName PortfolioDashboard` (PowerShell)
- 중지: 작업관리자 → "pythonw"·"cloudflared" 프로세스 종료
- 등록 해제: `Unregister-ScheduledTask -TaskName PortfolioDashboard -Confirm:$false`

만약 자동 시작이 안 되면 `run.bat`을 한 번 더블클릭해서 수동으로 켜면 됩니다.

## ☁️ 24시간 클라우드 (PC 꺼져도 폰에서 보기)

데이터 갱신은 PC에서, 보기는 폰에서 24시간 — 분리 구조.

**데이터 올리는 법 (PC에서)**:
1. RAW 파일을 `RAW DATA` 폴더에 떨어뜨림 (평소대로)
2. **`push_to_cloud.bat` 더블클릭** → 클라우드로 전송
3. 1~2분 후 폰에서 새로고침하면 최신 데이터 반영

**폰에서 보는 법**:
- Streamlit Cloud 주소(`https://...streamlit.app`)는 **고정**이라 한 번 즐겨찾기 하면 안 깨짐
- PC가 꺼져 있어도 항상 접속 가능 (읽기 전용 뷰)
- 같은 비밀번호로 입장

**구조**:
- `push_to_cloud.py` — RAW를 처리해 `cloud_data/portfolio_data.json` 생성 후 GitHub로 push
- Streamlit Cloud가 그 JSON을 읽어 표시 (`core/datasource.py`가 로컬/클라우드 자동 구분)
- 시장지표·뉴스·버블지표는 클라우드가 실시간으로 직접 갱신
- **저장소는 비공개(private)** — 자산 데이터 보호

## 📱 핸드폰·태블릿 접속 (PC 켜져 있을 때 — 보조)

PC가 켜져 있을 때 폰/태블릿에서도 같은 대시보드 접속 가능.

1. **PC 브라우저로 한 번 접속** → 사이드바 맨 위 **📱 외부 접속** 패널에 `https://*.trycloudflare.com` 주소 표시
2. 그 주소를 카카오톡 본인에게 보내거나 폰 브라우저에서 직접 입력
3. 같은 비밀번호 입력 → 동일 대시보드 사용
4. 폰 홈화면에 추가하면 앱처럼 사용 가능

**주의**:
- 외부 URL은 **PC를 끄거나 재부팅하면 새로 발급됨**. 사이드바에서 확인 후 폰 즐겨찾기 갱신
- PC가 꺼져 있으면 폰에서도 접속 불가 (서버가 PC에서 돔)
- 영구 고정 URL을 원하면 Cloudflare Named Tunnel (도메인 필요) 또는 Tailscale 추가 설정 가능

## 📅 데이터 갱신 (몇 초)

### 1️⃣ RAW 파일 떨어뜨리기 (메인 입력)

**위치**: `C:\Users\ksw96\Desktop\승환 자산 관리\RAW DATA\`

이 폴더에 **`자산 RAW DATA_YYYYMMDD_HHMM.xlsx`** 형식의 통합 자산 파일을 떨어뜨리면 끝.

폴더 안의 **가장 최근 수정된 .xlsx** 파일을 자동으로 사용합니다. 옛 파일을 그대로 둬도 무방 (이력 보존 가능).

**RAW 파일 컬럼 16개**:
| 기준일시 | 계좌번호 | 계좌별칭 | 계좌유형 | 구분 | 항목 | 티커 | 보유수량 | 매수평균가(원) | 현재가(원) | 매수금액(원) | 평가금액(원) | 평가손익(원) | 수익률(%) | 통화 | 비고 |

**구분** 값에 따라 자동 분류:
- `보유종목` → 실제 종목 (티커 필수, KRW 환산 완료된 평가금액 사용)
- `예수금` → 계좌별 현금 (음수면 자동 부채 처리)
- `매매손익` / `환차손익` / `계좌요약` / `전체요약` → 표시용, 자동 스킵

기존 신한 PDF·업비트 API·portfolio.xlsx의 coins/cash 시트는 더 이상 사용하지 않습니다.

### 2️⃣ 가족 자산 (`family` 시트) — 수기 입력, 마지막 값이 최신

`data/portfolio.xlsx`의 **`family`** 시트:

| 항목         | 금액         | 메모                  |
|-------------|--------------|----------------------|
| 승환 보증금   | 50000000     | 전세/월세 보증금 등   |
| 다은 보증금   | 30000000     |                      |
| 다은 투자자산 | 80000000     | 다은이 운용하는 투자자산 |

- 금액 입력 → 저장하면 그게 **최신 값**으로 고정 (마지막 입력만 반영)
- 행 추가도 자유 (예: "비상금", "부모님 자산" 등)
- 음수 입력 시 부채로 처리 (운용 총자산에서 차감)
- **운용 총자산 = 내 순자산(RAW) + 가족 자산** — 대시보드 상단에 별도 영역 표시

### 3️⃣ 종목 마스터 (`meta` 시트) — 선택, 한 번만

같은 엑셀 파일의 **`meta`** 시트에 종목별 메타정보:

| 티커  | 한글명     | 레버리지 | 자산버킷    | 목표비중(%) | 비고 |
|------|----------|--------|------------|-----------|------|
| TQQQ | 나스닥3X   | 3.0    | 빅테크      | 10        |      |
| SOXL | 반도체3X   | 3.0    | AI/반도체  | 15        |      |
| QQQ  | 나스닥100  | 1.0    | 빅테크      | 20        |      |
| ETH  | 이더리움   | 1.0    | 암호화폐    | 10        |      |

- **레버리지**: 일반=1.0, 2X ETF=2.0, 3X ETF=3.0, 인버스=-1.0
- **자산버킷**: 한글 자유 입력 (예: "빅테크", "방어주", "원자재", "암호화폐"…) — 같은 라벨끼리 자동 묶음
- **목표비중(%)**: 투자자산 합계 대비 원하는 비중 (예: 15 = 15%)
  - 입력하면 **리밸런싱 시그널** 패널이 활성화 — 현재 비중과 차이 계산, ±5%p 이탈 시 🔴매도/🟢매수 권장 금액 표시
  - 0(빈 값) → 목표 미설정으로 표시
- **안 채워도 OK**: 비어 있으면 `레버리지=1.0`, `자산버킷="미분류"`로 작동
- 한 번 입력해두면 RAW의 같은 티커에 자동 매칭됨

### 4️⃣ 대시보드 확인

`run.bat` 더블클릭 → 브라우저 자동으로 열림.

## 📈 이력은 어떻게 쌓이나? (중요)

| 파일 | 역할 | 사용자가 |
|------|------|----------|
| `RAW DATA/자산 RAW DATA_*.xlsx` | **오늘의 보유·예수금 통합** | 새로 받은 RAW를 폴더에 떨어뜨림 (최신 자동 사용) |
| `data/portfolio.xlsx`의 `meta` 시트 | **종목 마스터 (레버리지·자산버킷)** | 한 번 입력 후 신규 종목만 추가 |
| `data/history/snapshots.csv` | **누적 이력 (자동)** | **건드리지 마세요!** 자동 추가됨 |

**규칙**: `run.bat`을 켤 때마다 그 순간 평가금액·수익률을 자동으로 `snapshots.csv` 끝에 한 줄 추가해요. 같은 날 여러 번 켜면 마지막 값으로 덮어쓰기.

→ **매일 한 번씩 앱을 켜는 것만으로 자동 이력 관리**가 됩니다.
→ PDF나 엑셀을 바꿔도 이미 저장된 과거 스냅샷은 그대로 보존됩니다.
→ 누적이 5~7일 이상 되면 자동으로 추이 그래프가 그려져요.

⚠️ `snapshots.csv`를 직접 열어서 편집하지 마세요. 정말 초기화하고 싶으면 파일을 통째로 삭제하면 다음 실행 때 빈 상태로 다시 시작됩니다.

## ❓ 자주 겪는 문제

**Q. RAW 파일을 넣었는데 화면이 비어요**
A. 파일 위치(`C:\Users\ksw96\Desktop\승환 자산 관리\RAW DATA\`)와 헤더 16개 컬럼이 정확한지 확인. 사이드바에 RAW 파일명이 표시되는지 확인하세요.

**Q. 현재가가 0원이에요**
A. 인터넷 연결을 확인하세요. 사이드바의 "🔄 시세 새로고침"을 눌러보세요.

**Q. 환율이 안 나와요**
A. yfinance API 일시 오류일 수 있어요. 잠시 후 새로고침 해보세요.

**Q. RAW 파일을 수정해도 반영이 안 돼요**
A. 엑셀에서 파일을 **저장+닫고** 사이드바의 새로고침을 누르세요. 엑셀이 파일을 잡고 있으면 못 읽습니다.

**Q. 시계열 그래프가 안 보여요**
A. 데이터가 2건 이상 누적되어야 그래프가 그려져요. 며칠 동안 `run.bat`을 켜주세요.

**Q. 매일 자동으로 이력 저장하고 싶어요**
A. 클로드한테 "Windows 작업 스케줄러로 자동 캡처 만들어줘" 하면 됩니다.
